\# Dataset Access Instructions



This project uses datasets derived from LibriTTS and synthetic speech generated using Chatterbox-Turbo.



Due to the large size of the datasets and generated audio files, they are not included in this repository.



Dataset Components:



\* Original LibriTTS speech samples

\* Chatterbox-Turbo generated speech samples

\* Voice Conversion (VC) samples generated using Seed-VC

\* Emotional Voice Conversion (EVC) samples generated using Seed-VC



To reproduce the experiments:



1\. Download the LibriTTS dataset.

2\. Generate synthetic speech using Chatterbox-Turbo.

3\. Use the provided generation notebooks to create VC and EVC samples.

4\. Follow the AASIST training and evaluation pipeline provided in this repository.
